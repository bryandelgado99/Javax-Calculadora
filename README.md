## Conoce sobre el proyecto
Este proyecto implementa una calculadora, realizada en Java mediante la librería Swing GUI, que permite realizar las siguientes operaciones:
* Suma
* Resta
* Multiplicación
* División
* Potencia
* Raíz Cuadrada
* Función sen()
* Función cos()
* Función tan()
